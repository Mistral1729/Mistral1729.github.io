Sets, Maps and Structures

For our purposes, naive set theory as in [Halmos] will suffice. In fact, most working mathematicians not specialising in logic do not have to worry about axiomatic foundations. Logic is to mathematics what grammar is to languages, we use it intuitively all the time but we rarely care to dwell on it too much. In fact the naive set theory and basic propositional logic is in some sense a natural extension of our common sense, and thus rather intuitive. The purpose of rigour is not to banish intuition, but to banish bad intuition. Correct intuition is important, and rigour is there to helps us achieve that.
If it is at all possible to define the subject of algebra precisely, then it is the study of algebraic structures: sets on which operations are defined. But what is a set ?
We may call any collection of objects that is clearly determined a set. There are certain problems with this which we may conveniently ignore, at least for the time being. Each object in a set is called an element of said set. Usually we denote sets by uppercase letters and their elements by lowercase letters.
If "\(\mathcal{M}\) contains \(x\)" or "\(x\) is in \(\mathcal{M}\)" we write,
\[\begin{align} \mathcal{M} \ni x \ or \ x \in \mathcal{M}. \end{align}\]
If "\(\mathcal{M}\) doesn't contain \(y\)" or "\(y\) is not in \(\mathcal{M}\)" we write,
\[\begin{align} \mathcal{M} \not\ni y \ or \ x \notin \mathcal{M}. \end{align}\]
Okay, now what's an operation ? When talking about algebraic structures we really mean binary operations; an operation on the set \(\mathcal{M}\) is a map
\[\begin{align} \mathcal{M} \times \mathcal{M} \to \mathcal{M}, \end{align}\]
which is essentially a rule that assigns to every two elements of \(\mathcal{M}\) some element in the same set \(\mathcal{M}\).
The following number sets are well-known examples of algebraic structures, having the operations of addition and multiplication: \( \mathbb{N},\ \text{the set of all natural numbers}, \\ \mathbb{Z},\ \text{the set of all integers}, \\ \mathbb{Z}_{+} = \mathbb{N} \cup \{0\},\ \text{the set of all nonnegative integers}, \\ \mathbb{Q},\ \text{the set of all rational numbers}, \\ \mathbb{R},\ \text{the set of all real numbers}, \\ \mathbb{R}_{+},\ \text{the set of all nonnegative real numbers}. \) Not all number sets have addition or multiplication defined on them. Why ?
On the set of negative numbers multiplication is not defined, as the product of two negative numbers is positive. Moreover on the set of irrational numbers neither addition nor multiplication is defined, as the sum or product of two irrational numbers can be rational.

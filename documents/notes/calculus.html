Introduction

For this course we'll be following [Apostol] for calculus and analytic geometry. For the algebra we'll be using [Knapp] and [Vinberg].

Sets, Maps and Structures

For an axiomatic treatment of set theory refer to [Kunen]. For our purposes, naive set theory will suffice. In fact, most working mathematicians not specialising in logic do not have to worry about axiomatic foundations. Logic is to mathematics what grammar is to languages, we use it intuitively all the time but we rarely care to dwell on it too much. In fact the naive set theory and basic propositional logic is in some sense a natural extension of our common sense, and thus rather intuitive. The purpose of rigour is not to banish intuition, but to banish bad intuition. Correct intuition is important, and rigour is there to helps us achieve that.
If it is at all possible to define the subject of algebra precisely, then this is the study of algebraic structures: sets on which operations are defined. But what is a set ?
We may call any collection of objects that is clearly determined a set. There are certain problems with this: for example, can a set contain itself ?
Consider the set \(A_0\), the simplest possibility for which is for it to be empty i.e. \(A_0 = \{\}\) or \(\varnothing\). Clearly in this case its cardinality \(Card(A_0) = 0\). Let's now consider the set \(A_1 = \{A_0\} = \{\varnothing\}\). Note that \(Card(A_1) = 1\) and NOT 0 as it does contain exactly one element, \(\varnothing\).
In this way, by setting \(A_0 = \varnothing\) and \(A_{s(n)} = \displaystyle \bigcup_{k=0}^{n}A_k\), where \(s(n)\) is the successor of \(n\), we can define \(\mathbb{N}_0\) by
\(0 = Card(A_0) \in \mathbb{N}_0\), and
\(n = Card(A_n) \in \mathbb{N}_0 \Rightarrow s(n) = Card(A_s(n)) \in \mathbb{N}_0\).

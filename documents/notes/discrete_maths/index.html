1.1 Relations
Deﬁnition 1.1
A binary relation from set A to set B is any subset ρ ⊆ A × B.
An element a ∈ A is related t o b ∈ B in the relation ρ iﬀ (a, b) ∈ρ, which we denote
by a ρ b. If (a, b) /∈ρ then we write a ρ b.
Let A be a set. Then any subset ρ ⊆ A × A is a binary relation on A.
Deﬁnition 1.2
Let A be a set and let ρ be relation on A. Then ρ is said to be
1. reﬂexive iﬀ a ρ a ∀a ∈ A,
2. symmetric iﬀ a ρ b =⇒ b ρ a ∀a, b ∈ A,
3. transitive iﬀ a ρ b and b ρ c =⇒ a ρ c ∀a, b, c ∈ A,
4. irreﬂexive iﬀ a ρ a ∀a ∈ A,
5. asymmetric iﬀ a ρ b =⇒ b ρ a ∀a, b ∈ A,
6. intransitive iﬀ a ρ b and b ρ c =⇒ a ρ c ∀a, b, c ∈ A,
7. antisymmetric iﬀ a ρ b and b ρ a =⇒ a = b ∀a, b ∈ A,
and if ρ is all of (i), (ii) and (iii) then ρ is said to be an equivalence relation on
A.
If ρ is all of (i), (ii) and (vii) then ρ is said to be a partial order relation on A.
In the following six deﬁnitions ρ and σ are binary relations from A to B.
Deﬁnition 1.3
The complement of ρ is the relation ρ ⊆ A × B such that (aρb) ⇐⇒ (a ρ b) =
¬(a ρ b).
Deﬁnition 1.4
A relation on a set is an equivalence relation iﬀ it is reﬂexive, symmetric and
transitive.
Deﬁnition 1.5
A relation ⪯ on a set A is said to be a partial order relation, or simply a partial
order, iﬀ ⪯ is
1. reﬂexive ⇐⇒ (a ⪯ a ∀a ∈ A),
2. antisymmetric ⇐⇒ ((a ⪯ b) ∧ (b ⪯ a) =⇒ (a = b) ∀a, b ∈ A),
3. transitive ⇐⇒ ((a ⪯ b) ∧ (b ⪯ c) =⇒ (a ⪯ c) ∀a, b, c ∈ A)
on A.

2.1 Pigeon hole principle
Theorem 2.1 (Pigeonhole Principle)
If n pigeonholes are occupied by n + 1 or more pigeons, then at least one pigeonhole
is occupied by 2 or more pigeons.
Theorem 2.2 (Generalised Pigeonhole Principle)
If n pigeonholes are occupied by kn + 1 or more pigeons for k ∈ N, then at least one
pigeonhole is occupied by k + 1 or more pigeons.
